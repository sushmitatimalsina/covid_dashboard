# COVID Data Dashboard

A simple Python project that fetches COVID-19 data from an API, cleans the data, and visualizes the top countries by total COVID cases.

## Features

- Fetch COVID data using API
- Clean and analyze data using Pandas
- Visualize top countries using Matplotlib
- Sort countries by total cases

## Technologies Used

- Python
- Pandas
- Requests
- Matplotlib

## Installation

Install required libraries:

```bash
pip install pandas matplotlib requests
```

## Run the Project

```bash
python covid_dashboard.py
```

## Project Output

- Displays COVID statistics
- Shows top 10 countries by total cases
- Generates a bar chart visualization
